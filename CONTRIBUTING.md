# Contributing

Thank you for helping improve CONOW developer documentation!

## How to Contribute

1. **Fork** this repository and create a branch from `main`.
2. Make your changes — fix errors, add examples, or write new guides.
3. **Open a Pull Request** with a clear description of what you changed and why.

## Style Guide

- Write in English.
- Use [GitHub Flavored Markdown](https://github.github.com/gfm/).
- Hex values: uppercase, e.g. `0xA0`, `0x2710`.
- Register addresses: always show both decimal and hex.
- Frame bytes: space-separated uppercase hex, e.g. `A0 03 27 12 00 01 3E 6B`.

## Reporting Issues

Open an [Issue](../../issues) and include:
- Product model
- Firmware version (if known)
- Steps to reproduce
- Expected vs actual behavior
